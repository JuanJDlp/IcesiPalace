## Run Locally

Clone the project

```bash
  git clone https://github.com/Rony7v7/IcesiPalace
```

Go to the project directory

```bash
  cd IcesiPalace
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  cd ./icesipalace-frontend
  npm run start
```
